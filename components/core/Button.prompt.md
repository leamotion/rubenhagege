Pill-shaped button for primary actions and navigation links, in four palette variants.

```jsx
<Button variant="primary" size="lg" onClick={() => {}}>Passer commande</Button>
<Button variant="subtle" size="sm">Voir tout</Button>
```

Variants: `primary` (burgundy, main CTA), `dark` (ink, secondary CTA on light surfaces), `outline` (hairline border, transparent), `subtle` (soft dark pill, used for "Voir tout" links on dark sections). Sizes: `sm`, `md`, `lg`. Set `disabled` to gray it out.
