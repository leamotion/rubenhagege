Square icon button used for the "+" add-to-cart control and other single-glyph actions.

```jsx
<IconButton ariaLabel="Ajouter" icon={<PlusIcon />} variant="dark" />
```

Pass any inline SVG as `icon` (stroke-based, 1.5px, matches the brand's linear icon style). `variant="dark"` reproduces the dark square control seen on product rows.
